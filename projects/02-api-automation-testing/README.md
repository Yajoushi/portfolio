# API Automation Testing Suite

## Overview
A robust REST API testing framework demonstrating comprehensive QA expertise including API contract testing, load testing, and security validation.

## Technologies
- **Framework**: RestAssured / Postman / Thunder Client
- **Language**: Java / JavaScript
- **Test Runner**: JUnit 5 / Mocha
- **API Testing**: REST Assured / Axios
- **Contract Testing**: PACT
- **Security Testing**: OWASP validation
- **CI/CD**: GitHub Actions

## Key Features
✅ CRUD operations testing  
✅ Request/response validation  
✅ Authentication & authorization testing  
✅ Error code validation  
✅ API contract testing  
✅ Rate limiting tests  
✅ Security headers validation  
✅ Performance benchmarking  

## Test Coverage
- Positive test cases
- Negative test cases
- Boundary value analysis
- Equivalence partitioning
- Security testing (SQL injection, XSS)
- Rate limiting & throttling

## Endpoints Tested
- User Management API
- Product Catalog API
- Order Processing API
- Payment API
- Authentication API

## How to Run
```bash
# Install dependencies
npm install  # or maven

# Run all API tests
npm test

# Run with coverage
npm run test:coverage

# Performance testing
npm run test:performance

# Security scan
npm run test:security
```
